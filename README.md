# Fog_Forecast
KOPIS 날씨 빅데이터 분석, 안개 발생 여부 진단 모델 개발

이 프로젝트는 기상 데이터를 기반으로 안개 발생 여부를 예측하는 모델을 개발하는 것을 목표로 합니다. 

주요 단계는 다음과 같습니다:

1. 데이터 EDA 및 전처리
2. 모델 학습
3. 모델 평가
